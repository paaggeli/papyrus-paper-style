Ancient Paper Texture

I created this Ancient Paper Texture using ComfyUI and followed this YouTube tutorial on how to create patterns and tiles.
I'm including the JSON file containing the ComfyUI workflow, so you can easily replicate similar results with your own projects.

I created this papyrus-style texture specifically for my upcoming project, Tartarus Insight, which I'm building in public.

To create the rough edges

To create the rough edge at the top, I started with a pattern I made in ComfyUI, papyrus.jpg. I opened this image in Photopea, cropped the bottom section to a height of 10px, and saved it as papyrus-top.png. Then, using Photopea's Eraser Tool, I gently removed some color from the top of papyrus-top.png to create a rough edge effect. Since papyrus.jpg is a repeating pattern, the bottom of papyrus-top.png will seamlessly blend with papyrus.jpg when used together in our HTML.

Similarly, for the left edge, I cropped the right section of papyrus.jpg to a width of 10px and removed color from the left side to create the rough edge effect.

To create the right and bottom edges, I used a different approach. For the bottom edge, I created a new transparent image with the same width as papyrus.jpg and a height of 10px. I filled in with the background color I plan to use for the div.papyrus-content element and then used the Eraser Tool to create rough edges.

Similarly, for the right edge, I created a transparent image with the same height as papyrus.jpg and a width of 10px, filled in with the same background color, and used the Eraser Tool to create rough edges.
